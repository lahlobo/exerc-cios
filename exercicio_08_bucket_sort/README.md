# Exercício 08: Bucket Sort

Este exercício implementa o Bucket Sort para ordenar:
- Números em ponto flutuante no intervalo [0, 1).
- Números inteiros positivos em intervalos maiores.

## Como funciona
- **Ponto flutuante:** Os números são distribuídos em baldes baseados em sua posição relativa no intervalo [0, 1).
- **Números inteiros:** Os números são distribuídos em baldes com intervalos definidos por `bucket_size`.
